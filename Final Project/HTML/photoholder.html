Skewers Photo Holder

What you need:

  • Paper
  • Skewers
  • Glue gun
  • Pair of Scissors
  • Wooden clips


Procedure

  1. Create a grid like frame on the base paper using the skewers
  2. Connect the skewers using a glue gun
  3. Add some color and design.
  4. Cutout pictures of your choice and place it using the clips.
